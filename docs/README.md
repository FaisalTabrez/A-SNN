## Where to add new docs

- Use this folder for newcomer guides, roadmap documents, and repo-wide notes.
- Use `gen5/docs/` for technical Gen-5 architecture, runbooks, and backend
  migration notes.
- Use `gen5/outputs/<run-name>/analysis.md` for analysis tied to one experiment
  bundle.
